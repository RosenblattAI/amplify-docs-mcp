AWS Amplify Documentation MCP Server

This is a test page for the AWS Amplify Documentation MCP Server. This server provides tools for searching and reading AWS Amplify documentation using the Model Context Protocol (MCP).

About

This MCP server is configured to search the AWS Amplify documentation from the official GitHub repository. It uses the Probe search engine to provide fast and relevant search results.

Configuration

{
  "gitUrl": "https://github.com/aws-amplify/docs.git",
  "gitRef": "main",
  "autoUpdateInterval": 60,
  "toolName": "search_amplify_docs",
  "toolDescription": "Search AWS Amplify documentation using the probe search engine."
}
